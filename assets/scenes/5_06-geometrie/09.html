Le plan cartésien : les quatre quadrants
Nous avons vu au niveau 4 comment repérer un point dans le premier quadrant du plan cartésien. Quand un des éléments d’un couple (x,y) est négatif, ce dernier ce retrouve soit dans le deuxième, le troisième ou le quatrième quadrant.
Exemple : Le point (-4, 6) se retrouve dans le deuxième quadrant
Le point (2, -2) se retrouve dans le quatrième quadrant
Le point (-3, -5) se retrouve dans le troisième quadrant
Afin de faciliter la localisation de points dans le plan, les mathématiciens ont décidé d’adopter la convention suivante :
Distance horizontale
  • Lorsqu’une distance horizontale (x) est orientée vers la droite, elle est positive.
  • Lorsqu’une distance horizontale (x) est orientée vers la gauche, elle est négative.
Distance verticale
  • Lorsqu’une distance verticale (y) est orientée vers le haut, elle est positive.
  • Lorsqu’une distance verticale (y) est orientée vers le bas, elle est négative.
Exemple : À l’aide de ces conventions, nous pouvons déterminer les coordonnées des points A, B, C et D sur le graphique ci-dessous.
Le point A est déterminé par sa distance horizontale (sur l’axe des x) : +2 et sa distance verticale (sur l’axe des y) : +3.
On représente le point A par le couple ordonné (2, 3).
De même, le point B est déterminé par sa distance horizontale (sur l’axe des x) : -3 et sa distance verticale (sur l’axe des y) : -2½.
On représente le point B par le couple ordonné (-3, -2½).
La distance horizontale (sur l’axe des x) du point C est +4 et sa distance verticale (sur l’axe des y) est –1.
On représente le point C par le couple ordonné (4, -1).
La distance horizontale (sur l’axe des x) du point D est –2 et sa distance verticale (sur l’axe des y) est 0.
On représente le point D par le couple ordonné (-2, 0).
