Les étapes de l'addition de plusieurs nombres positifs et négatifs
Pour additionner des nombres positifs et négatifs :
  • On élimine les parenthèses
  • On effectue les opérations
Exemple : Effectuer : 3 + (-4) + (-2) + (5)
Solution : 3 + (-4) + (-2) + (5)
= 3 - 4 - 2 + 5 ⟵ on élimine les parenthèses
= 2 ⟵ on effectue les opérations

Remarques
Les données d'un problème peuvent être présentées de différentes façons.
  • Elles peuvent contenir des parenthèses pour chaque terme sauf le premier.
    -3 + (5) + (-2) + (3) + (-6) + (-5)
  • Elles peuvent ne contenir des parenthèses que pour les nombres négatifs.
    -3 + 5 + (-2) + 3 + (-6) + (-5)
  • Elles peuvent ne pas contenir de parenthèses.
    -3 + 5 - 2 + 3 - 6 - 5
