Addition et soustraction de nombres entiers
L'addition de nombres entiers par déplacements sur la droite numérique
Pour mieux comprendre les déplacements sur une droite numérique, nous allons nous servir de la droite numérique verticale. En effet, la droite numérique peut être horizontale ou verticale.
Exemple 1 : Additionner : 3 + (-5) + (7)
Imaginez que cette droite numérique est un thermomètre.
  • D’abord, il faisait 3 °C et la température a baissé de 5 degrés; ce qui fait -2 °C.
  • La 1ere flèche indique une baisse de 5 degrés; on l'exprime : -5.
  • On a donc : 3 - 5 = -2
  • Ensuite, la température a monté de 7 degrés, elle est donc passé de -2 °C à +5°C
  • La 2e flèche indique une hausse de 7 degrés; on l’exprime : +7.
  • On a donc : -2 + 7 = 5
Alors, 3 + (-5) + (7) = 5
Exemple 2 : Additionner : 3 + (-5) + (7) sur la droite numérique horizontale.
3 + (-5) + (7) = 5
Pour additionner un nombre négatif : le déplacement se fait vers la gauche.
3 + (-5) = -2
Pour additionner un nombre positif : le déplacement se fait vers la droite.
-2 + 7 = 5
Alors, 3 + (-5) + (7) = 5
Exemple 3 : Additionner par déplacements : -3 + (-2) + (8)
-3 + (-2) = -5
-5 + 8 = 3
Alors, -3 + (-2) + (8) = 3
