Addition et soustraction de polynômes
L’addition de deux polynômes
Étapes :
  1. éliminer les parenthèses
  2. grouper les monômes (ou termes) semblables
  3. simplifier
Exemple A:
(2x^2 + 3x - 5) + (5x^2 - x + 2)
Solution :
= 2x^2 + 3x - 5 + 5x^2 - x + 2
éliminer les parenthèses
= 2x^2 + 5x^2 + 3x - x - 5 + 2
grouper les termes semblables
= 7x^2 + 2x - 3
additionner les termes
On peut aussi placer les polynômes verticalement pour faire l’opération.
Exemple B:
(3a5 + 2a4 + 2a3 - 5a2) + (2a5 + a4 + 3a3 + 6a2)
Solution :
3a^5 + 2a^4 + 2a^3 - 5a^2 2a5 + a^4 + 3a^3 + 6a^2 5a^5 + 3a^4 + 5a^3 + a^2
Exemple C.
(6x^4 + 4x^3 - x + 6) + (- 2x^4 + 2x^2 + 2x)
Solution :
6x^4 + 4x^3 - x + 6
- 2x^4 + 2x^2 + 2x
4x^4 + 4x^3 + 2x^2 + x + 6
On remarque que dans l’exemple C, on a laissé des espaces là où des monômes correspondants manquaient. Si ces espaces portent à confusion, on peut aussi écrire des polynômes ayant pour coefficient zéro, comme dans l’exemple suivant où on reprend les mêmes termes de l’exemple C en ajoutant des zéros.
6x4 + 4x3 + 0x2 - x + 6
- 2x4 + 0x3 + 2x2 + 2x + 0
4x4 + 4x3 + 2x2 + x + 6
