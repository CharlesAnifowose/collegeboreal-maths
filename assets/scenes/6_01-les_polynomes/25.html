La résolution de problèmes de nombres entiers consécutifs
Certains problèmes renferment des nombres consécutifs comme données. Par nombres consécutifs, on entend des nombres qui se suivent tels que 1, 2, 3 ou 110, 111, 112; ces nombres ont une différence de un entre eux. Pour représenter un ensemble de nombres consécutifs sous une forme symbolique, on attribue la valeur «x» au premier nombre. Comme le deuxième nombre d’un ensemble de nombres consécutifs représente un de plus que le premier, on lui attribue la valeur de x + 1. Comme le troisième nombre d’un ensemble de nombres consécutifs représente deux de plus que le premier, on lui attribue la valeur de x + 2. Alors 4, 5 et 6 sont les 3 nombres consécutifs.
Exemple A Trouvez 3 nombres consécutifs dont la somme est 36. 1) Comprendre le problème Trouvez 3 nombres consécutifs dont la somme est 36. 2) Élaborer un plan choisir la variable : soit x le premier nombre soit x + 1 le deuxième nombre soit x + 2 le troisième nombre bâtir l’équation : x + (x + 1) + (x + 2) = 36 3) Exécuter le plan résoudre l’équation : x + (x + 1) + (x + 2) = 36 x + x + 1 + x + 2 = 36 x + x + x = 36 - 1 - 2 3x = 33 x = 11 Donc x = 11, soit le premier nombre. x + 1 = 11 + 1 = 12, soit le deuxième nombre. x + 2 = 11 + 2 = 13, soit le troisième nombre. Donc les 3 nombres consécutifs sont 11, 12 et 13. 4) Vérifier l’ensemble-solution 3 nombres consécutifs dont la somme est 36. 11 + 12 + 13 = 36 36 = 36
