La résolution de problèmes de nombres consécutifs pairs
Il arrive que certains problèmes renferment des nombres consécutifs pairs tel 2, 4, 6 ou 52, 54, 56. Comme il y a une différence de 2 entre les nombres consécutifs pairs, on attribue la valeur de x au premier, x + 2 au deuxième et x + 4 au troisième.
Exemple A Trouver quatre nombres consécutifs pairs dont la somme est 148. 1) Comprendre le problème Trouver quatre nombres consécutifs pairs dont la somme (addition) est 148. 2) Élaborer un plan choisir la variable : soit x le premier nombre soit x + 2 le deuxième nombre soit x + 4 le troisième nombre soit x + 6 le quatrième nombre bâtir l’équation : x + (x + 2) + (x + 4) + (x + 6) = 148 3) Exécuter le plan résoudre l’équation : x + (x + 2) + (x + 4) + (x + 6) = 148 x + x + 2 + x + 4 + x + 6 = 148 x + x + x + x = 148 - 2 - 4 - 6 4x = 136 x = 34 Donc x = 34, soit le premier nombre. x + 2 = 34 + 2 = 36, soit le deuxième nombre x + 4 = 34 + 4 = 38, soit le troisième nombre x + 6 = 34 + 6 = 40, soit le quatrième nombre Donc, les quatre nombres consécutifs sont 34, 36, 38 et 40. 4) Vérifier l’ensemble-solution La somme de 4 nombres consécutifs est 148. 34 + 36 + 38 + 40 = 148 148 = 148
