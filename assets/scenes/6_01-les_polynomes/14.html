Élever à une puissance On dit élever à une puissance un binôme ou polynôme lorsqu’on veut que celui-ci soit multiplié par lui-même un certain nombre de fois. On exprime une puissance par un exposant. Écrire une expression avec un exposant est une façon raccourcie d’écrire une multiplication.
Exemple C Élever à la puissance de 2, le binôme (a + b) (a + b)2 = (a + b) ( a + b) Étapes : 1- (a) (a) = a2 2- (a) (b) = ab 3- (b) (a) = ba = ab 4- (b) (b) = b2 Donc (a + b)2 = (a + b) ( a + b) = a2 + ab + ab + b2 = a2 + 2ab + b2
