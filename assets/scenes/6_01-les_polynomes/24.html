La résolution de problèmes
Il va sans dire que l’on doit lire attentivement toutes les données d’un problème avant de le résoudre.
Exemple A Lucienne a un certain nombre de timbres. Pauline en a 8 fois autant. Si les deux en possèdent 180, combien chacune en possède-t-elle? 1) Comprendre le problème On veut déterminer combien chacune ont de timbres. Pauline en a 8 fois plus que Lucienne. La somme de tous les timbres est de 180. 2) Élaborer le plan choisir la variable : soit y le nombre de timbres de Lucienne soit 8y le nombre de timbres de Pauline (8 fois le nombre de Lucienne) bâtir l’équation : y + 8y = 180 3) Exécuter le plan résoudre l’équation : y + 8y = 180 9y = 180 y = 20 Donc, y = 20, soit le nombre de timbres de Lucienne. 8y = 8 x 20 = 160, soit le nombre de timbres de Pauline. Donc, Lucienne a 20 timbres et Pauline en a 160. 4) Vérifier l’ensemble-solution vérifier : Lucienne possède 20 timbres. Pauline en a 8 fois autant, soit 160. Les deux en possèdent 180, soit 160 + 20.
Exemple B Sylvain a 16 ans de plus que Stéphane et la somme de leurs âges est 48. Quelle est l’âge de chacun? 1) Comprendre le problème L’âge de Sylvain plus l’âge de Stéphane est égale à 48. Sylvain à 16 ans de plus que Stéphane. 2) Élaborer le plan choisir la variable : soit y l’âge de Stéphane Donc y + 16 est l’âge de Sylvain bâtir l’équation : y + (y + 16) = 48 3) Exécuter le plan résoudre l’équation : y + (y + 16) = 48 y + y + 16 = 48 2y = 48 - 16 2y = 32 y = 16 Donc, y = 16, soit l’âge de Stéphane. y + 16 = 16 + 16 = 32, soit l’âge de Sylvain. Donc, Stéphane a 16 ans et Sylvain a 32 ans. 4) Vérifier l’ensemble-solution Stéphane a 16 ans; Sylvain a 16 ans de plus, soit (16 + 16) 32 ans; la somme de leurs âges, soit 16 + 32, est 48. Remarquez : Dans l’exemple B, il y a deux inconnues. Quand il y a deux inconnues, on attribue le y à l’inconnue à laquelle l’autre inconnue est comparée. Puisque l’âge de Sylvain est comparé à l’âge de Stéphane, on a attribué le y à l’âge de Stéphane.
Exemple C Un grille-pain et une bouilloire coûtent 54$. Le grille-pain coûte 6$ de plus que la bouilloire. Trouvez le coût de chacun de ces appareils. 1) Comprendre le problème Le prix d’un grille-pain plus le prix d’une bouilloire égale 54$. Le grille-pain coûte 6$ de plus que la bouilloire. 2) Élaborer le plan choisir la variable : soit x le coût de la bouilloire soit x + 6 le coût du grille-pain bâtir l’équation : x + (x + 6) = 54 3) Exécuter le plan résoudre l’équation : x + (x + 6) = 54 x + x + 6 = 54 2x = 54 - 6 2x = 48 x = 24 Donc, x = 24, soit le coût de la bouilloire. x + 6 = 24 + 6 = 30, soit le coût du grille-pain. Donc, la bouilloire coûte 24$ et le grille-pain coûte 30$ 4) Vérifier l’ensemble-solution Une bouilloire coûte 24$. Un grille-pain coûte 6$ de plus, soit 24$ + 6$ = 30$. Les deux appareils coûtent 54$, soit 24$ + 30$. Remarquez : Dans l’exemple C, le prix du grille-pain est comparé au prix de la bouilloire; c’est pourquoi on a attribué le «x» au prix de la bouilloire.
Exemple D La somme de deux nombres est 23. Si l’on additionne le plus petit nombre à 3 fois le plus grand, la somme est 55. Quels sont les deux nombres? 1) Comprendre le problème 2) Élaborer le plan choisir la variable : soit x le plus petit nombre soit (23 - x) le plus grand nombre (Pour réussir à trouver 23 - x, on emploie la donnée du problème qui dit que la somme de deux nombres est 23. Puisqu’un des nombres est x, on obtient l’expression le plus petit + le plus grand = 23 ou x + le plus grand = 23 donc le plus grand = 23 - x soit 3 (23 - x) trois fois le plus grand nombre bâtir l’équation : x + 3 (23 - x) = 55 3) Exécuter le plan résoudre l’équation : x + 3 (23 - x) = 55 x + 69 - 3x = 55 x - 3x = 55 - 69 -2x = -14 x = 7 Donc, x = 7, soit le plus petit nombre. 23 - x = 23 - 7 = 16, soit le plus grand nombre. Donc, 7 est le plus petit nombre et 16 est le plus grand nombre. 4) Vérifier l’ensemble-solution La somme de deux nombres est 23, soit 7 + 16. Si l’on additionne le plus petit nombre, soit 7, à 3 fois le plus grand, c’est-à-dire 3 x 16 ou 48, on obtient 55, soit 7 + 48 = 55.
Exemple E Aline, Bobby et Corine ont 1 236$. Aline à deux fois l’argent de Bobby et Corine a 150$ de plus qu’Aline et Bobby ensemble. Combien chacun a-t-il d’argent? 1) Comprendre le problème Combien chacun a-t-il d’argent? Remarquer que l’argent n’est pas divisé également entre les trois individus. 2) Élaborer un plan choisir la variable : - soit x l’argent de Bobby (on attribue x à Bobby, puisqu’on attribue le x à l’inconnue à laquelle les autres inconnues sont comparées. Aline est comparé à Bobby. Corine est comparé à Aline et Bobby. Donc les inconnues Aline et Corine sont comparées à Bobby) - soit 2x l’argent d’Aline - soit (x + 2x) + 150 l’argent de Corine bâtir l’équation : x + 2x + (x + 2x) + 150 = 1 236 3) Exécuter le plan résoudre l’équation : x + 2x + (x + 2x) + 150 = 1 236 x + 2x + x + 2x + 150 = 1 236 x + 2x + x + 2x = 1 236 - 150 6x = 1 086 x = 181 Donc x = 181, soit l’argent de Bobby. 2x = 181 × 2 = 362, soit l’argent d’Aline (x + 2x) + 150 = 181 + 2 (181) + 150 = 181 + 362 + 150 = 693, soit l’argent de Corine Donc, Aline a 362$, Bobby a 181$ et Corine a 693$ 4) Vérifier l’ensemble-solution Aline a deux fois l’argent de Bobby, soit 2 fois 181$ soit 362$. Corine a 150$ de plus que Aline qui a 362$ et Bobby qui a 181$; 150$ + 362$ + 181$ = 693$. Aline, Bobby et Corine ont 362$ + 181$ + 693$ pour un total de 1 236$.
Exemple F Pierre s’achète des souliers et une chemise. Les souliers coûtent le double de ce que coûte la chemise. Si la facture s’élève à 112,50$, combien coûtent les souliers et combien coûte la chemise? 1) Comprendre le problème Il faut déterminer le coût des souliers et le coût de la chemise. Les souliers coûtent le double de ce que coûte la chemise. 2) Élaborer un plan choisir la variable : soit y le prix de la chemise soit 2y le prix des souliers bâtir l’équation : ______________________________ 3) Exécuter le plan résoudre l’équation : ______________________________ Donc, y = _______$, soit le prix de la chemise. 2y = 2 × _____$ = _____$, soit le prix des souliers. Donc, la chemise coûte 37,50$ et les souliers coûtent 75,00$. 4) Vérifier l’ensemble-solution La chemise coûte 37,50$. Les souliers coûtent le double de ce que coûte la chemise, soit 2 × 37,50$ = 75,00$. Les deux articles coûtent 112,5
Exemple G Francine veut acheter des oranges et des pamplemousses pour ces déjeuners. Cette semaine, chez l’épicier le plus près, les oranges et les pamplemousses coûtent le même prix. Elle décide d’acheter 8 oranges et 3 pamplemousses. Si sa facture est de 2,75$ , quel est le prix de ces fruits ? Combien a-t-elle dépensé pour les oranges ? 1) Comprendre le problème On cherche premièrement le prix des fruits. Deuxièmement, on veut savoir quelle somme Francine dépense pour les oranges seulement. Première question Quel est le prix des fruits ? 2) Élaborer un plan Si y est l’inconnue représentant le prix des fruits, alors on peut exprimer le problème comme suit : 8y + 3y = 2,75$ 􀃎 8 oranges fois le prix (y), plus 3 pamplemousses fois le prix (y) qui est le même que celui des oranges 3) Exécuter le plan 8y + 3y = 2,75$ 11y = 2,75$ y = 0,25$ Chaque fruit coûte 0,25$. (On peut aussi exprimer cette somme comme suit 25¢) Deuxième question Combien a-t-elle dépensé pour les oranges ? 2) Élaborer un plan Sachant maintenant que chaque fruit coûte 0,25$, on peut faire une deuxième équation afin de déterminer le montant dépensé pour les oranges. 8 (0,25$) = z z représentant le montant dépensé pour les oranges 3) Exécuter le plan 8 (0,25$) = z 2$ = z Francine à donc dépensé 2$ pour les oranges seulement. 4) Vérifier la solution 8(0,25$) + 3(0,25$) = 2,75$ 2,00$ + 0,75$ = 2,75$
Exemple H Les apprenants du programme ont monté une pièce de théâtre qui sera présentée dans une salle de spectacle pouvant accueillir 190 personnes. Les billets sont en vente depuis une semaine pour la somme de 5$. On a déjà amassé 675$ en vendant des billets à l’avance. Combien de billets a-ton vendus et combien d’autres peut-on vendre avant que la salle soit pleine ? 1) Comprendre le problème On pose deux questions dans le problème. La première est de trouver le nombre de billets qui ont étés vendu à l’avance. La deuxième est de déterminer le nombre de billets non vendus. Première question Trouver le nombre de billets qui sont déjà vendus 2) Élaborer un plan 5 b = 675 5$ fois le nombre de billets (b) donne le montant de 675$ 3) Exécuter le plan 5 b = 675 b = 675 ÷ 5 b = 135 On a vendu 135 billets à l’avance. Deuxième question Combien y a-t-il de billets non vendus? 2) Élaborer un plan B = 190 - 135 B représente le nombre de billets non vendus. 3) Exécuter le plan B = 190 - 135 B = 55 Il reste 55 billets à vendre afin de remplir la salle. 4) Vérifier l’ensemble-solution 135 (billets vendus) + 55 (billets à vendre) = 19
Exemple I Dès l’automne, un jardinier s’apprête à planter ses tulipes. Il a 64 bulbes en tout. Il veut en mettre 22 devant sa maison et diviser le reste également dans ses trois plates-bandes derrière la maison. Combien de bulbes seront plantés dans chacunes des plates-bandes derrière la maison ? 1) Comprendre le problème 22 bulbes seront plantés devant le maison et le reste sera divisé également dans les trois plates-bandes arrières. 2) Élaborer un plan 64 = 22 + x + x + x x représente le nombre de tulipes dans chaque plate bande derrière la maison. 3) Exécuter le plan 64 = 22 + 3x 42 = 3x 42 = x 3 14 = x x = 14 Le jardinier pourra mettre 14 bulbes dans chacune des plates-bandes arrières. 4) Vérifier l’ensemble-solution 64 = 22 + 3(14) 64 = 22 + 42 64 = 64
Exemple J Un nombre est multiplié par 5. Au résultat, on ajoute le triple de ce nombre. On obtient 96. Quel est ce nombre ? 1) Comprendre le problème Déterminer le nombre manquant 2) Élaborer un plan Si on représente le nombre par la lettre a. Un nombre (a) est multiplié par 5 5a Au résultat, on ajoute le triple de ce nombre + 3a On obtient 96 = 96 3) Exécuter le plan 5a + 3a = 96 8a = 96 a = 96 ÷ 8 a = 12 4) Vérifier la solution 5(12) + 3(12) = 96 60 + 36 = 96 96 = 96
