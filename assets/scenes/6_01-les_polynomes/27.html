La résolution de problèmes de nombres consécutifs impairs
Il arrive que certains problèmes renferment des nombres consécutifs impairs tels 3, 5, 7 et 71, 73, 75. Comme il y a une différence de 2 entre les nombres consécutifs impairs, on attribue la valeur de x au premier, x + 2 au deuxième et x + 4 au troisième. Remarquez que c’est la même méthode que pour les nombres pairs. Si le premier nombre de la série est pair, alors la suite est composée de nombres pairs. Si le premier nombre de la série est impair, alors la suite est composée de nombres impairs.
Exmemple A Trouver 3 nombres consécutifs impairs dont la somme est 225. 1) Comprendre le problème Trouver 3 nombres consécutifs impairs dont la somme est 225. 2) Élaborer le plan choisir la variable : soit x le premier nombre soit x + 2 le deuxième nombre soit x + 4 le troisième nombre bâtir l’équation : x + (x + 2) + (x + 4) = 225 3) Exécuter le plan résoudre l’équation : x + (x + 2) + (x + 4) = 225 x + x + 2 + x + 4 = 225 x + x + x = 225 - 2 - 4 3x = 219 x = 73 Donc x = 73, soit le 1er nombre. x + 2 = 73 + 2 = 75, soit le 2e nombre x + 4 = 73 + 4 = 77, soit le 3e nombre Donc, les nombres sont 73, 75 et 77. 4) Vérifier l’ensemble-solution La somme de 3 nombres consécutifs impairs est 225. 73 + 75 + 77 = 225 225 = 225
Exmemple B Un automobiliste parcourt 2 240 kilomètres en 4 jours. À chaque jour, il fait 96 kilomètres de moins que la veille. Combien de kilomètres parcourt-il à chaque jour? 1) Comprendre le problème Déterminer le nombre de kilomètres parcouru à chaque jours. 2) Élaborer un plan choisir la variable : soit x le nombre de km du 1er jour soit x - 96 le nombre de km du 2e jour soit x - 96 - 96 ou x - 192 le nombre de km du 3e jour soit (x - 192) - 96 ou x - 288 le nombre de km du 4e jour 1er jour + 2e jour + 3e jour + 4e jour = 2 240 km bâtir l’équation : x + (x - 96) + (x - 192) + (x - 288) = 2 240 km 3) Exécuter le plan résoudre l’équation : x + (x - 96) + (x - 192) + (x - 288) = 2 240 x + x - 96 + x - 192 + x - 288 = 2 240 x + x + x + x = 2 240 + 96 + 192 + 288 4x = 2 816 x = 704 Donc, x = 704, soit le nombre de km du 1er jour x - 96 = 704 - 96 = 608, soit le nombre de km du 2e jour x -192 = 704 - 192 = 512, soit le nombre de km du 3e jour x - 288 = 704 - 288 = 416, soit le nombre de km du 4e jour Donc, les nombres sont 704, 608, 512 et 416. 4) Vérifier l’ensemble-solution L’automobiliste fait 96 km de moins à chaque jour; il fait 2 240 km en 4 jours. 704 + 608 + 512 + 416 = 2 240 2 240 = 2 240
