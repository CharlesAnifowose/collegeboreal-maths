Soustraction de deux polynômes
Étapes :
  1. éliminer les parenthèses
  2. grouper les monômes (ou termes) semblables
  3. simplifier
Exemples A:
(2x^2 + 3x - 5) - (5x^2 - x + 2)
Attention : Le signe devant la deuxième parenthèse étant «-», les signes à l’intérieur de celle-ci changent lorsqu’on élimine les parenthèses.
Solution:
= 2x^2 + 3x - 5 - 5x^2 + x - 2 éliminer les parenthèses
= 2x^2 - 5x^2 + 3x + x - 5 - 2 grouper les termes semblables
= -3x^2 + 4x - 7 additionner les termes
Exemples B:
Si l’on veut soustraire le polynôme -5a^5 + 4a^4 - 4a^3 du polynôme 10a^5 - 6a^4 - 2a^3, on peut disposer les termes verticalement puisque c’est une disposition plus pratique.
Solution:
10a^5 - 6a^4 - 2a^3
- ( -5a^5 + 4a4 - 4a3 )
Ensuite, on doit changer les signes des termes dans la parenthèse, c’est-àdire trouver l’opposé du polynôme à soustraire.
10a^5 - 6a^4 - 2a^3
5a^5 - 4a^4 + 4a^3
15a^5 - 10a^4 + 2a^3
Observation : On inscrit habituellement la réponse en suivant l’ordre décroissant des puissances (exposants) des variables et les nombres s’inscrivent à la fin.
Exemples :
15x^6 + 12x^8 - 6x^4 + 3x^5 - 17 devient 12x^8 + 15x^6 + 3x^5 - 6x^4 - 17
8b^3c - 9b^2c^2 - 10bc^3 - 10b^4c^3 + 6b^6 devient 6b^6 - 10b^4c^3 + 8b^3c - 9b^2c^2 - 10bc^3
