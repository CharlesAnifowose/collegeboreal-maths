Les quatre opérations de base sur les nombres naturals
Les nombres naturels
Les nombres que nous utilisons tous les jours, par exemple pour compter, s'appellent nombres naturels.
Les nombres naturels comprennent 0 et tous les autres nombres entiers plus grands que 0 c'est-à-dire 1, 2, 3, 4, ...jusqu'à l'infini.

Les trois petits points, que l'on appelle points de suspension, nous indiquent que les nombres continuent jusqu'à l'infini.
On peut représenter les nombres naturels sur une droite.
Cette droite se nomme droite numérique. La pointe de flèche nous indique qu'on peut énumérer les nombres jusqu'à l'infini.
