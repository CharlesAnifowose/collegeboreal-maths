Construire des modèles de figures à trois dimensions
Pour vous aider à visualiser les figures à trois dimensions, nous vous proposons de construire différentes figures à partir de gabarits. On peut tracer ces gabarits sur des feuilles de papier ou préférablement sur une feuille de carton rigide.
Le cube : Tracez ce gabarit sur une feuille de papier puis découpez-le. Pliez ensuite le long des lignes pointillées et collez les côtés avec du ruban adhésif.
La pyramide à base carrée : Tracez ce gabarit sur une feuille de papier ou de carton puis découpez-le. Pliez ensuite le long des lignes pointillées et collez les côtés avez du ruban adhésif.
Le cylindre : Tracez ce gabarit sur une feuille de papier ensuite découpez-le. Formez un tube en pliant chaque petit rectangle le long des lignes pointillées. Les deux bouts devraient se rejoindre. Collez ces bouts à l'aide d'un morceau de ruban adhésif. Pliez les deux cercles le long de la ligne pointillée. Les deux cercles devraient recouvrir les côtés du tube. Fixez les deux cercles aux côtés à l'aide de ruban adhésif.
Pour visualiser un objet, il n’est pas toujours nécessaire de construire un modèle à partir de matériel concret. Dessiner la figure sur un géoplan ou sur une feuille de papier quadrillé peut tout aussi bien faire l’affaire.
Le papier quadrillé
Étape 1 : On trace le rectangle qui deviendra la face avant du prisme en suivant les lignes qui forment le quadrillage.
Étape 2 : On trace trois lignes parallèles qui s'étendent vers l'arrière plan.
Étape 3 : En suivant les lignes du quadrillage, on rejoint les trois lignes parallèles de point à point.
Notez que l’on peut rejoindre les sommets par des lignes pointillées. Ceci donne l’illusion que le prisme est transparent.
On peut aussi tracer la figure précédente en respectant les étapes suivantes :
Étape 1 : On trace le rectangle qui deviendra la face du prisme.
Étape 2 : On trace le deuxième rectangle qui deviendra la face arrière.
Étape 3 : On rejoint les sommets des deux rectangles.
Notez que les étapes de chacune de ces méthodes peuvent être répétées pour d'autres solides à faces planes, telles que le cube et tous les prismes droits ou obliques.
