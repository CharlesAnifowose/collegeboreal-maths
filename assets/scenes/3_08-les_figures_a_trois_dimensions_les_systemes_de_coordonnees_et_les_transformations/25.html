Les transformations de figures à deux dimensions :
les réflexions, les translations et les rotation
Les réflexions
L’orsqu’une figure, dont les sommets correspondants sont situés à égale distance de l’axe de symétrie se transforme en une figure identique, on dit qu’elle a subi une réflexion.
Une réflexion est soit :
  • l’image réfléchie d'un objet comme si on le regardait dans un miroir
  • la réflexion d'un objet lorsqu'un miroir est placé sur son axe de symétrie
