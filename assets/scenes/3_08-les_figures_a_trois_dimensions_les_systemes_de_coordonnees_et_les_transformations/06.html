Classifier les polyèdres
Pour classifier les figures géométriques, on compare les caractéristiques propres à chaque figure.
Ainsi, pour distinguer entre eux les polyèdres, on compare les caractéristiques suivantes :
  1. le nombre de faces
  2. le nombre d'arêtes
  3. le nombre de sommets
  4. le nombre et le type d'angles
Examinons chacune de ces caractéristiques.
1. Les faces
Les faces sont les figures géométriques qui forment les polyèdres. Un polygone plat s'appelle une face.
Par exemple, un prisme triangulaire est composé de 5 faces : trois rectangles et deux triangles.

Rappel: Les polyèdres sont des solides qui sont limités par des polygones plats. Le cube, les prismes et les pyramides sont des polyèdres. Les cônes, les sphères et les cylindres ne sont pas des polyèdres.
2. Les arêtes
Le segment formé par l'intersection ou la rencontre de deux faces planes se nomme arête.
Le prisme suivant est composé de 6 faces et de 12 arêtes.
Voici d'autres exemples d'arêtes.
3. Le sommet
Un sommet est :
  • le point de rencontre de deux côtés d'un polygone
    • ou
  • le point de rencontre des arêtes d'un solide.
4. L'angle
Un angle est formé par la rencontre de deux côtés ou de deux arêtes. Cet angle peut être soit un angle droit, un angle aigu ou un angle obtus.
Rappel:
  • Un angle droit mesure 90°
  • Un angle aigu est plus petit que 90°
  • Un angle obtus est un angle dont la mesure est comprise entre 90° et 180°
