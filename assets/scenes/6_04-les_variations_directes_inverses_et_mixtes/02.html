Les variations directes, inverses et mixtes
Le rapport
Les éléments d’un problème peuvent souvent s’écrire sous forme de rapport lorsqu’on fait une comparaison entre ceux-ci. Par exemple, comparons le nombre d’oranges dans une salade de fruits par rapport au nombre de pommes. Si la recette indique cinq oranges et trois pommes, alors le rapport est de 5 à 3. Les rapports peuvent s’écrire de trois façons : 5 à 3 ou 5:3 ou 5/3
Lorsqu’on exprime un rapport, le premier nombre que l’on indique se rapporte à la première quantité mentionnée dans le texte. Par exemple si on dit que le rapport d’oranges et de pommes est de 5 à 3, le 5 qui apparaît le premier dans le rapport, représente les oranges qui apparaissent en premier dans le texte. Également, le 3 qui apparaît en deuxième lieu dans le rapport représente les pommes qui apparaissent en deuxième lieu dans le texte.
Autre exemple :
Le rapport d’hommes et de femmes dans une classe est de 12 à 17. L’élément hommes apparaît en premier lieu dans le texte, donc le premier nombre du rapport (12) représente les hommes. Également, l’élément femmes apparaît en deuxième lieu dans le texte, donc le deuxième nombre (17) représente les femmes.
