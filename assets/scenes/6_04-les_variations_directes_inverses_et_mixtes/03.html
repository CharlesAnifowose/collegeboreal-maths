RAPPORT ENTRE LES INCONNUES D’UNE ÉQUATION
Il est parfois possible d’établir le rapport existant entre deux inconnues d’une équation donnée. La condition essentielle toutefois est que la valeur absolue de l’équation égale zéro.
Voici un exemple.
Établir le rapport existant de x à y à partir de l’équation 3x - 4y = 0.
Solution :
1.
Isolons le terme contenant x dans le membre gauche de l’équation.
Puisque 3x - 4y = 0
3x = 4y
2.
Résolvons pour x. Si 3x = 4y
x = (4y)/3
3.
Comparons x à y en remplaçant x par sa valeur, soit (4y)/3.
Ainsi, x/y = (4y/3)/y
