Méthode 3 : Résolution de l’équation quadratique par la formule quadratique
Les mathématiciens ont développé une formule dont le but est de résoudre les équations quadratiques.
Cette formule est :
Elle porte le nom de formule quadratique. Les lettres a et b représentent les coefficients de l’équation quadratique ax^2 + bx + c = 0;
ainsi,
a est le coefficient de x^2
 
b est le coefficient de x
et
c est le terme connu de l’expression.
Dans la formule quadratique, la valeur de b^2 - 4ac qui est sous le radical, nous fournit des indications sur les racines de l’équation et est appelée le discriminant de l’équation.
Lorsque b^2 - 4ac > 0, nous obtenons 2 racines réelles et distinctes;
Lorsque b^2 - 4ac = 0 nous obtenons 1 seule racine;
Lorsque b² - 4ac < 0 il n’y a pas de racines réelles; on dit alors que les racines sont imaginaires.
(On peut aussi représenter cette radicande par le symbole ∆, qui est la lettre grecque delta).
