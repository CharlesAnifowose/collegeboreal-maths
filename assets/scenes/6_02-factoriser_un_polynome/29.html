Exemple :
Résoudre 4x^2 - 5x = 6
Solution:

  1. Organisons l’équation de façon à ce que tous les termes du membre gauche égalent zéro.

    4x^2 - 5x - 6

  2. Factorisons le côté gauche de l’équation.

    Il s’agit de trouver 2 termes dont le produit est -24 et dont la somme égale le coefficient du terme moyen, soit -5 dans cet exemple.
    4x^2 - 5x - 6 = 0
    (4x^2 - 8x) + (3x - 6) = 0
    4x(x - 2) + 3(x - 2) = 0
    (x - 2)(4x + 3) = 0

  3. Supposons chaque facteur est égal à zéro et résolvons.

    Si x - 2 = 0
    Si 4x + 3 = 0
    x = 2
    4x = -3
     
    x = -3/4
    L’ensemble-solution est {2, -3/4}.
