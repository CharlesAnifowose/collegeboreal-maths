Un autre exemple :
C. Résoudre 4x^2 - 16 = 0
Solution:

  1. Nous remarquons que le membre gauche de cette équation se compose d’une différence de carrés; trouvons les facteurs de cette différence de carrés.

    4x^2 - 16 = 0
    (2x + 4) (2x - 4) = 0

  2. Puisque le côté droit de l’équation égale zéro, au moins un des facteurs doit être égal à zéro.

    Si 2x + 4 = 0
    Si 2x - 4 = 0
    2x = -4
    2x = 4
    x = -4/2
    x = 4/2
    x = 2
    x = -2
    L’ensemble-solution est {-2, 2}.
    Effectuez la vérification afin de vous assurer que les deux racines satisfaissent l’équation.
