ÉQUATION QUADRATIQUE
On appelle équation quadratique, ou du second degré, une équation de la forme ax^2 + bx + c = 0 où 2 est la plus haute puissance de l’inconnue.
Dans une telle équation, les lettres a et b représentent des coefficients alors que c est une valeur connue.
Exemples d’équations quadratiques :
x^2 + 14x + 13 = 0
 
4x^1 - 5x - 6 = 0
 
2x^2 - 9x + 7 = 0
De la première exemple :
Nous retrouvons deux sortes d’équations quadratiques à une inconnue :
les équations quadratiques incomplètes
et
les équations quadratiques complètes
L’équation quadratique complète se présente sous la forme générale ax^2 + bx + c = 0. Nous y retrouvons toujours des termes en x^2, en x et en c.
Si l’équation ne renferme pas de termes en x ou en c, l’équation est alors dite incomplète.
Ainsi, ax^2 + c = 0 est une équation quadratique incomplète parce qu’il manque le deuxième terme bx. De même, ax^2 + bx = 0 est une équation quadratique incomplète parce qu’il manque le terme c.
