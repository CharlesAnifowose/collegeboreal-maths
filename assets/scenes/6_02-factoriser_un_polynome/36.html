B. Résoudre.
Solution:

  1. Isolons le radical qui nous semble le plus compliqué.

  2. Élevons les deux membres de l’équation au carré.

  3. Isolons le radical restant.

  4. Élevons les deux membres de l’équation au carré.

    Puisque

  5. Résolvons l’équation.

    36x - 36 = 100 - 40x + 4x^2
    -4x^2 + 76x - 136 = 0
    Puisque 4 est un facteur commun et que nous voulons rendre le coefficient de x² positif, simplifions en divisant l’équation par -4.
    x^2 - 19x + 34 = 0
    Par factorisation, nous obtenons :
    x^2 - 19x + 34 = 0
    (x - 17) (x - 2) = 0
    Si x - 17 = 0
    Si x - 2 = 0
    x = 17
    x = 2
    Les racines sont 17 et 2.

  6. Ces deux racines satisfont-elles toutes deux l’équation donée?

    Substituons dans l’équation donnée x = 17 et résolvons.
    Comme le membre gauche n’égale pas le membre droit, 17 est une racine à rejeter.
    Substituons dans l’équation l’autre racine, x = 2.
    Comme le membre droit égale celui de gauche, 2 est la seule racine de l’équation donnée.
