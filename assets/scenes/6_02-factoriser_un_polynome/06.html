  4. Ainsi, dans l’expression (a + b) m + (a + b) n, le facteur commun est a + b.
    Pour décomposer l’expression (a + b) m + (a + b) n en mettant en évidence le
    facteur commun, soit (a + b), on divise l’expression par (a + b).
    (a + b) m_(a+b) + (a + b) n_(a + b) = m + n
    On inscrit ensuite le facteur commun, soit (a + b) suivi du quotient entre
    parenthèses, soit (m + n). On obtient donc (a + b) (m + n).
    Vérification : a + b
    × m + n_ am + bm
    + an + bn_ am + bm + an + bn est équivalent à (a + b) m + (a + b) n
