Équation quadratique irrationnelle
Une équation quadratique irrationnelle est une équation quadratique dont l'une des variables est sous radical.
Pour résoudre une telle équation, il s’agit d’isoler le radical d’un côté de l'équation; on élève ensuite les deux membres de l'équation à une puissance égale à l'indice du radical et on résout l'équation.
Voici un exemple.
A. Résoudre.
Solution:

  1. Isolons le radical d’un côté de l'équation.

  2. Élevons les deux côtés de l'équation au carré.

  3. Résolvons l'équation. On doit premièrement additionner les termes semblables.

    x + 4 = 4x^2 - 28x + 49
    x + 4 - 4x^2 + 28x - 49 = 0
    -4x^2 + 29x - 45 = 0
    ou 4x^2 - 29x + 45 = 0

  4. Appliquons la formule quadratique. Nous avons a = 4, b = -29 et c = 45.

Dans l'équation donnée, substituons x par chacune de ces deux valeurs et résolvons.
Comme le membre de droite n'égale pas le membre de gauche, 9/4 est une racine à rejeter.
