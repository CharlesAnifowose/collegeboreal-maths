Voici un exemple.
A. Résoudre 4x^2 - 4x = 0
Solution:

  1. Dans cette équation, nous remarquons que 4x est un facteur commun du membre gauche; effectuons donc la mise en évidence de ce facteur commun.

    4x^2 - 4x = 0
    4x(x-1) = 0

  2. Puisque le côté droit de l’équation est zéro, au moins un des facteurs doit être égal à zéro. Alors on doit déterminer les valeurs que x pourrait prendre.

    Si x - 1 = 0
    Si 4x = 0
    x = 1
    x = 0
    L’ensemble-solution est {1, 0}.
Notons qui’il est impossible d’extraire la racine carrée d’un nombre négatif. Une telle racine est dite «imaginaire».
Encore une fois l’ensemble-solution contient deux racines possibles. Effectuons la vérification afin de s’assurer que les deux racines satisfaissent l’équation.
Si x = 1
Si x = 0
4x^2 - 4x = 0
4x^2 - 4x = 0
4(1)^2 - 4(1) = 0
4(0)^2 - 4(0) = 0
4 - 4 = 0
0 - 0 = 0
0 = 0
0 = 0
Dans les deux cas, il y a égalité entre chacun des membres de l’équation; les deux racines satisfont donc l’équation.
