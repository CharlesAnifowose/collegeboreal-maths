Voici un exemple :
B. Résoudre 3x^2 + 5x = -4
Solution:

  1. Retranscrivons l’équation de sorte que tous les termes du côté gauche de l’équation égalent zéro.

    3x^2 + 5x = -4

  2. Identifions les valeurs de a, de b et de c.

    Nous avons a = 3, b = 5 et c = 4.

  3. Substituons ces valeurs dans la formule et résolvons.

Puisque le ∆ < 0, les racines de l’équation ne sont pas réelles.
(∆ = discriminant)
L’ensemble-solution est
Ces racines portent le nom d’imaginaires car elles contiennent la racine carrée d’un nombre négatif.
