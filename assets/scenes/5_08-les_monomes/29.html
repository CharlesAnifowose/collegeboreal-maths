Les équations contenant des fractions ou des nombres décimaux
Exemple 1 :
7/8 = a + 1/4
a + 1/4 = 7/8
Notez : On a changé le membre de gauche pour le membre de droite afin d’avoir la variable dans le membre de gauche. Cette étape n’est pas nécessaire, cependant pour certaines personnes il est plus facile d’isoler la variable si celle-ci est dans le membre de gauche. Remarquez qu’on devrait obtenir le même résultat si on gardait la variable dans le membre de droite.
a = 7/8 - 1/4       On transpose la fraction 1/4 . Elle devient (-1)/4 dans le membre de droite.
a = 7/8 - 2/8       On trouve un dénominateur commun afin de faire la soustraction des fractions.
a = 5/8
