Exemples :
a) 5^3 signifie 5*5*5 qui est égale à 125
b) a^8 signifie a*a*a*a*a*a*a*a
c) (3b)^4 signifie 3b * 3b * 3b * 3b
l’exposant 4 affecte toute la parenthèse
d) 3b^4 signifie 3 * b * b * b * b
l’exposant 4 affecte le b seulement ; le 3 n’est pas affecté par l’exposant
e) (-2)^3 = -2 * -2 * -2
l’exposant 3 affecte toute la parenthèse
f) -2^3 = -2*2*2
l’exposant 3 affecte le 2 seulement ; le négatif n’est pas affecté par l’exposant
