LA MÉTHODE DE TRANSPOSITION
II est possible d'appliquer les règles précédentes de façon plus rapide en faisant une transposition. On transpose un terme d'un membre en inscrivant son oppose dans l'autre membre.
Les termes qui sont additionnés à l'inconnue ou soustraits de l'inconnue
Si un terme est additionné dans un membre de l’équation, il est soustrait lorsqu’on le transpose dans l'autre membre de l’équation et vice versa.
additionné dans le membre gauche
soustrait dans le membre gauche
→ soustrait dans le membre droit
→ additionné dans le membre droit
Les termes qui multiplient l'inconnue
Un terme qui est multiplié a l'inconnue, divise le membre de l'autre côté de l'équation.
multiplié dans le membre gauche → divisé dans le membre droit
Les termes qui divisent l'inconnue
Le terme qui divise l'inconnue est multipliée lorsqu'il est transposé dans l'autre membre de l’équation.
divisé dans le membre gauche → multiplié dans le membre droit
Raccourci : Lorsqu'on transpose une fraction qui multiplie l'inconnue, on multiplie l'autre membre de l’équation par la réciproque de cette fraction.
