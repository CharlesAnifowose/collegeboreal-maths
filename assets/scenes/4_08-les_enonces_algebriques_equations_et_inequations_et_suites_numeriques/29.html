LA MÉTHODE DE TRANSPOSITION
Les termes qui sont additionnes à l'inconnue OU soustraits de l'inconnue
Si un terme est additionne dans un membre de !l’équation, ii est soustrait lorsqu’on le transpose dans l'autre membre de l’équation et vice versa.
additionné dans le membre gauche -> soustrait dans le membre droit
soustrait dans le membre gauche -> additionné dans le membre droit
Les termes qui multiplient l'inconnue
Un terme qui est multiplie a l'inconnue, divise le membre de l'autre cote de l'équation.
multiplié dans le membre gauche -> divisé dans le membre droit
Les termes qui divisent I 'inconnue
Le terme qui divise l'inconnue est multipliée lorsqu'il est transpose dans l'autre membre de l’équation.
divisé dans le membre gauche -> multiplié dans le membre droit
Raccourci : Lorsqu'on transpose une fraction qui multiplie l'inconnue, on multiplie l 'autre membre de l’équation par la réciproque de cette fraction.
