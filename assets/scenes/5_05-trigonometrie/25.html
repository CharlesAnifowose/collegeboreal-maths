Disons que l’on choisit de trouver le côté c avec le rapport cos 60 ° = b/(4,5)

On a:

b = cos 60 * (4,5)

b = 2,25 cm ≈ 2,3 cm

Il nous reste à trouver le côté a. Nous avons 3 solutions possibles :
  1. utiliser le théorème de Pythagore
  2. lerapport sin60° = a/(4,5)
  3. le rapport cos 30 ° = a/(4,5)
Choisissons le théorème de Pythagore :
a = sqrt(h^2 - b^2) = sqrt(4,5^2 - 2,3^2)
a ≈ 3,9cm
Nous avons trouvé :
∠ A = 60 °
a = 3,9cm
b = 2,3cm
