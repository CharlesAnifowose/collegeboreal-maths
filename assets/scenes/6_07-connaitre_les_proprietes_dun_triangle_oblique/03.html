Construire le croquis d’un triangle à l’aide des données d’un problème
Lorsqu’on demande de résoudre un triangle oblique, on donne certaines informations par rapport à ce triangle, et on doit déterminer les autres données. Les problèmes de triangles obliques peuvent représenter plusieurs cas;
A. Illustrez le triangle ayant les données suivantes :
l’angle A = 35°
l’angle B = 100°
le côté b = 8 cm
Il faut s’assurer de bien illustrer les angles. Un angle de 100° est un angle obtus et doit être ouvert (plus grand qu’un angle droit). Également, l’angle de 35° est un angle aigu et doit être fermé (plus petit qu’un angle droit).
B. Illustrez le triangle ayant les données suivantes :
a = 9 mètres
b = 12 mètres
c = 15 mètres
Lorsqu’on illustre ce triangle, il faut s’assurer que le côté c est le plus grand, et le côté a, le plus petit. On place ensuite les lettres majuscules (qui représentent les angles) opposées à leur côté respectif.
C. Illustrez le triangle ayant les données suivantes :
B = 30°
b = 27cm
c = 36cm
Il faut s’assurer que le côté c est plus grand que le côté b lorsqu’on illustre le triangle. Également, l’angle de 30o est relativement petit.
