Le périmètre du parallélogramme
Pour calculer le périmètre d'un parallélogramme, on additionne la
longueur des deux bases* (b) et la longueur des deux côtés courts (c).
P = b + b + c + c
P = 2b + 2c
ou
où b est la base et c est le côté court.
La base est habituellement le côté sur lequel repose la figure; c’est
souvent le côté long.
