Notez : Certaines calculatrices ne sont pas vraiment programmées pour faire de l’algèbre. Un petit test pour vérifier est :
La calculatrice devrait afficher (- 4) comme réponse.
Si votre calculatrice affiche ( -2) ou ‘ error ‘ c’est peut-être qu’elle n’est pas dans le bon ‘mode’ ou tout simplement qu’elle n’est pas programmée pour faire de l’algèbre. Vous pouvez essayer de mettre un signe de multiplication devant la parenthèse : 2 * (1 - 3) =. Vous vous rendrez compte qu’il est probablement plus facile d’effectuer les calculs sans calculatrice. Il est quand même bon de faire des expériences afin de maîtriser l’utilisation de votre calculatrice. Mais vous n’aurez pas droit à la calculatrice pour écrire le test.
