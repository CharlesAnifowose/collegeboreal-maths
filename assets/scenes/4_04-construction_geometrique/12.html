Exemple 2 : Déterminer si les proportions du tracteur ont été respectées.
Échelle : 1 cm = 75 cm
Le rapport entre les longueurs est : 300: 225: 150 = 4: 3: 2.
Le rapport entre les hauteurs est : 300: 225: 150 = 4: 3: 2 .
Les rapports sont les mêmes pour les longueurs ainsi que pour les hauteurs.
Les proportions ont été respectées.
