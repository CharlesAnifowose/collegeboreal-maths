Nous pouvons tracer d’autres coordonnées pour le graphique en substituant d’autres valeurs pour x.
Ainsi, si x = 5
y = x^2 - 2x - 8
y = (5)^2 - 2(5) - 8
y = 25 - 10 - 8
y = 7
Nous avons les coordonnées (5, 7); en reflétant ce point de l’autre côté de l’axe de symétrie, nous obtenons (-3, 7).
Voici un autre exemple :
Tracer le graphique de la fonction y = -2x^2 + 12x - 10 en établissant :
  1. les coordonnées du sommet
  2. les abscisses à l’origine
  3. l’ordonnée à l’origine.
Dans cette exemple, le coefficient de x^2 est négatif, alors le sommet est maximum puisque la courbe s’ouvre vers le bas.
Solution :
  1. Pour trouver les coordonnées du sommet, déterminons l’axe de symétrie.
    Nous avons a = -2 et b = 12.
    Puisque x = -b/2a
    x = -12/(2(-2))
    x = -12/-4
    x = 3
    L’axe de symétrie est à x = 3.
    Pour obtenir la valeur de y, remplaçons x par 3 dans l’équation.
    y = -2x^2 + 12x - 10
    y = -2(3)^2 + 12(3) - 10
    y = -18 + 36 - 10
    y = 8
    Les coordonnées du sommet sont (3, 8).
    Puisque le coefficient de x^2 est négatif, le graphe s’ouvrira vers le bas. Aussi, le sommet sera le point maximum du graphique.
  2. Nous trouvons les abscisses à l’origine en cherchant les racines de l’équation.
    -2x^2 + 12x - 10 = 0
    Afin de rendre le coefficient de x^2 positif, multiplions l’équation par
    (-1) (-2x^2 + 12x - 10) = (-1)(0)
    2x^2 - 12x + 10 = 0
    Puisque 2 est un facteur commun à cette expression, simplifions celleci par 2.
    Nous obtenons 2 (x^2 - 6x + 5) = 0.
    Par factorisation, nous obtenons : 2 (x - 5) (x - 1) = 0
    À ce point, on cherche les valeurs de x pour lesquels les parenthèses sont égales à zéro.
    Si (x - 5) = 0
    x = 5
    Si (x - 1) = 0
    x = 1
    Les abscisses à l’origine sont 5 et 1. Nous avons donc les coordonnées (5, 0) et (1, 0).
  3. Cherchons l’ordonnée à l’origine en remplaçant x dans l’équation par zéro.
    y = -2x^2 + 12x - 10
    y = -2 (0)^2 + 12(0) - 10
    y = -10
    L’ordonnée à l’origine est -10; nous avons ainsi les coordonnées (0, -10).
    En reflétant ce point de l’autre côté de l’axe de symétrie, nous obtenons les coordonnées (6, -10).
