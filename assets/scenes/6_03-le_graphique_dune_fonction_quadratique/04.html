Nous remarquons dans le graphique ci-haut que la courbe traverse l’axe des x à -1 et à 3. Ces deux valeurs sont les racines de l’équation, elles sont aussi appelées les abscisses à l’origine; ce sont les valeurs de x lorsque y égale zéro. Nous obtenons ces racines en résolvant l’équation à l’aide de l’une des méthodes que nous avons vues précédemment, soit : par factorisation, en complétant le carré ou à l’aide de la formule quadratique.
Nous remarquons aussi que la courbe traverse l’axe des y à -3. Cette valeur est appelée l’ordonnée à l’origine; c’est la valeur de y lorsque x = 0.
Il n’est donc plus nécessaire de choisir des valeurs arbitraires pour construire le graphique d’une équation quadratique. En effet, si nous cherchons les coodonnées du sommet, les racines de l’équation et l’ordonnée à l’origine, nous pourrons tracer ce graphique sans peine.
Voici un exemple :
Tracer le graphique de la fonction y = x^2 - 2x - 8.
Solution :
  1. Pour trouver les coordonnées du sommet, déterminons d’abord l’axe de symétrie.
    Nous avons : a = 1 et b = -2
    Puisque:
    x = -b/2a
    x = (-(-2))/(2(1))
    x = 2/2 = 1
    L’axe de symétrie est à x = 1.
  2. Pour obtenir la valeur de y, remplaçons x par 1 dans l’équation.
    Soit
    y = x^2 - 2x - 8
    y = (1)^2 - 2(1) - 8
    y = 1 - 2 - 8
    y = -9
    Les coordonnées du sommet sont (1, -9).
  3. Nous trouvons les abscisses à l’origine en cherchant les racines de l’équation.
    x^2 - 2x - 8 = 0
    Par factorisation, nous obtenons : (x - 4) (x + 2) = 0
    Pour trouver les racines, il s’agit de trouver les valeurs qui vérifient l’équation.
    Remarquer que si une des parenthèses est égale à zéro, alors l’équation est égale à zéro, (0) (x + 2) = 0 ou (x - 4) (0) = 0 parce que la multiplication par zéro donne toujours zéro.
    Premières parenthèses : On cherche la valeur pour laquelle on obtient zéro, donc
    (x - 4) = 0
    x = 4
    Si on vérifie, lorsque x égale 4, l’équation égale zéro.
    (x - 4) (x + 2) = 0
    (4 - 4) (4 + 2) = 0
    (0) (6) = 0
    Deuxièmes parenthèses : On cherche la valeur pour laquelle on obtient zéro, donc
    (x + 2) = 0
    x = -2
    Si on vérifie, lorsque x égale -2, l’équation égale zéro.
    (x - 4) (x + 2) = 0
    (-2 - 4) (-2 + 2) = 0
    (-6) (0) = 0
    Donc, les abscisses à l’origine sont 4 et –2.
    Puisque les abscisses à l’origine sont les valeurs de x lorsque y égale zéro, nous avons les coordonnées (4, 0) et (-2, 0).
  4. Cherchons l’ordonnée à l’origine en remplaçant x par zéro dans l’équation.
    y = x^2 - 2x - 8
    y = (0)^2 - 2(0) - 8
    y = -8
    L’ordonnée à l’origine est -8.
    Puisque l’ordonnée à l’origine est la valeur de y lorsque x égale 0, nous avons les coordonnées (0, -8).
  5. En reflétant ce point de l’autre côté de l’axe de symétrie nous obtenons les coordonnées (2, -8), les deux points étant situés à la même hauteur et, une distance égale les sépare de chaque côté de l’axe de symétrie.
  6. Plaçons ces couples dans le plan cartésien.
